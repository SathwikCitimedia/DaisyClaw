# Northflank

Deploy DaisyClaw on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy DaisyClaw](https://northflank.com/stacks/deploy-daisyclaw) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy DaisyClaw now**.
4. Set the required environment variable: `DAISYCLAW_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the DaisyClaw template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the DaisyClaw service.
8. Open the public DaisyClaw URL at `/daisyclaw` and connect using the configured shared secret. This template uses `DAISYCLAW_GATEWAY_TOKEN` by default; if you replace it with password auth, use that password instead.

## What you get

- Hosted DaisyClaw Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so `daisyclaw.json`,
  per-agent `auth-profiles.json`, channel/provider state, sessions, and
  workspace survive redeploys

## Connect a channel

Use the Control UI at `/daisyclaw` or run `daisyclaw onboard` via SSH for channel setup instructions:
