# DaisyClaw Amazon Bedrock Mantle Provider

Official DaisyClaw provider plugin for routing Amazon Bedrock Mantle models through OpenAI-compatible provider flows.

Install from DaisyClaw:

```bash
daisyclaw plugin add @daisyclaw/amazon-bedrock-mantle-provider
```

Use this plugin when your Bedrock deployment exposes Mantle-compatible model routing and you want DaisyClaw agents to address those models through the Bedrock Mantle provider.
