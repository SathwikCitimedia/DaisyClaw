# DaisyClaw Codex

Official DaisyClaw provider and harness plugin for OpenAI Codex app-server integration. It exposes the Codex-managed GPT model catalog and the Codex runtime surfaces used by DaisyClaw agents.

Install from DaisyClaw:

```bash
daisyclaw plugin add @daisyclaw/codex
```

Use this plugin when you want DaisyClaw to run Codex-backed model turns, media understanding, and prompt overlays through the Codex app-server harness.
