# DaisyClaw Feishu/Lark

Official DaisyClaw channel plugin for Feishu and Lark workplace chats. Community maintained by @m1heng.

Install from DaisyClaw:

```bash
daisyclaw plugin add @daisyclaw/feishu
```

Configure the Feishu/Lark app credentials in DaisyClaw, then connect the plugin to the chats where agents should receive and send messages.
